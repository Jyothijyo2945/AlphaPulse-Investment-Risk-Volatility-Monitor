# AlphaPulse – Investment Risk & Volatility Monitor

AlphaPulse is a financial analytics project built using Python and Power BI to monitor portfolio risk, market volatility, and forecast future portfolio performance.

## Project Objective
- Analyze stock market data  
- Calculate portfolio risk (VaR)  
- Monitor volatility and correlations  
- Forecast future portfolio outcomes  

## Tech Stack
- Python (Pandas, NumPy)  
- yfinance API  
- Power BI  
- Monte Carlo Simulation  

## Project Workflow

### Week 1 – Data Collection
- Collected stock data using yfinance  
- Selected 10 stocks + S&P 500  
- Cleaned and prepared dataset  

### Week 2 – Analysis
- Calculated log returns  
- Rolling volatility (30-day)  
- Correlation matrix  
- Value at Risk (VaR)  
- Monte Carlo simulation  

### Week 3 – Dashboards
- Market Overview  
- Portfolio Performance  
- Risk & Volatility Monitor  
- What-If Analysis  
- Forecast & Simulation  

### Week 4 – Finalization
- Executive Summary dashboard  
- Automation using Python script  
- Benchmark validation  

## Dashboard Pages
- Market Overview  
- Portfolio Performance  
- Risk & Volatility Monitor  
- What-If Scenario  
- Forecast & Simulation  
- Executive Summary  

## Automation
- Python script updates market data  
- CSV files refresh automatically  
- Power BI dashboard updates on refresh
